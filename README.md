Single-file landing page for Mindvalley University 2026 in Tallinn, Estonia · July 20 – August 2, 2026.

**Live:** https://marisha-mv.github.io/mvu-2026-landing/

## Stack

- Single static `index.html` — no build step
- Google Sans Flex (Mindvalley brand font, via Google Fonts)
- Live countdown to early bird deadline (edit `EARLY_BIRD_END` in `<script>` at bottom)
- Real MV press logos, speaker photos, attendee portraits pulled from storyblok-cdn.mindvalley.com

## Editing

Open `index.html` in any browser to preview. To change:
- Countdown deadline: `EARLY_BIRD_END` in the `<script>` block
- Ticket pricing: search `1,599` / `2,149` / `7,995`
- YouTube aftermovie: search for `youtube.com/embed/`
- Press logos: inside `<section class="press">`
